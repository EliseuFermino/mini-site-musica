# Guia de Fluxo Git

Este documento explica o fluxo Git utilizado no projeto **Mini Site Musical**.

O objetivo é manter um processo simples, seguro e profissional para toda alteração feita no projeto.

## Visão geral do fluxo

Toda alteração deve seguir esta sequência:

```text
main atualizada
→ nova branch
→ alteração no código
→ commit
→ push
→ Pull Request
→ Code Review
→ aprovação
→ merge na main
→ pull local
→ exclusão da branch local
