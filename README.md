# Mini Site Musical

Projeto de estudo criado para praticar Git, GitHub e fluxo profissional de desenvolvimento usando tarefas, branches, commits, Pull Requests e revisão de código.

## Objetivo do projeto

O objetivo deste projeto é simular um ambiente real de desenvolvimento, onde existem dois papéis:

* Diretor de Desenvolvimento
* Desenvolvedor Júnior

Mesmo sendo uma simulação feita pela mesma pessoa, o projeto segue um fluxo profissional de trabalho.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Git
* GitHub
* Visual Studio
* ClickUp

## Fluxo de trabalho

O fluxo utilizado neste projeto será:

1. Criar tarefa no ClickUp
2. Mover a tarefa para `IN PROGRESS`
3. Criar uma branch no GitHub
4. Fazer as alterações no código
5. Criar um commit
6. Enviar a branch para o GitHub
7. Abrir um Pull Request
8. Revisar o Pull Request
9. Fazer merge na branch `main`
10. Mover a tarefa para `DONE` no ClickUp

## Padrão de branches

Exemplos de nomes de branches:

```text
docs/criar-readme-inicial
feat/criar-estrutura-html
feat/criar-cabecalho-site
style/aplicar-layout-inicial
fix/corrigir-botao-ouvir
deploy/publicar-github-pages
```

## Padrão de commits

Exemplos de mensagens de commit:

```text
docs: cria README inicial do projeto
feat: cria estrutura inicial HTML
feat: adiciona cabeçalho do site
style: aplica layout inicial com CSS
fix: corrige texto do botão ouvir
deploy: publica site no GitHub Pages
```
